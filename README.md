# Sistema de Suporte ao Cliente

Projeto desenvolvido em Python com interface de terminal.

## Funcionalidades

* Lista de produtos
* Consulta de endereço
* Consulta de horário de funcionamento
* Falar com atendente
* Encerramento do programa
* Cores no terminal

## Tecnologias

* Python 3
* os
* time

## Como executar

```bash
python main.py
```
